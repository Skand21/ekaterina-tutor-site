# Блок для Tilda: материалы, отзывы и текст

Готовый файл для вставки в Tilda:

- `ekaterina-materials-reviews-tilda.html`

Как вставить:

1. Открыть страницу в Tilda.
2. Добавить блок `T123 HTML-код` в нужное место страницы.
3. Вставить внутрь весь код из файла.
4. Сохранить и опубликовать страницу.

Что внутри:

- блок про материалы и систему занятий;
- карточки с отзывами и результатами;
- текст от первого лица Екатерины;
- CTA-блок с Telegram и каналом;
- адаптивная верстка для телефона.

Важно: блок не использует скриншоты и не раскрывает личные данные учеников.
