# Сайт Екатерины Песни

Статический лендинг репетитора по математике.

## Что внутри

- `index.html` - структура страницы.
- `styles.css` - визуальный стиль и адаптив.
- `script.js` - меню и копирование текста для обращения.
- `assets/` - безопасные публичные изображения из предоставленных материалов.
- `AUDIT.md` - аудит лендинга и вопросы для следующей итерации.
